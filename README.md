## Configuración de Desarrollo

### Prerequisitos

- Instalar [Node.js] que incluye [Node Package Manager][npm]
- Instalar [Visual Studio Code][vcode], para editar nuestro código (User Installer version)

### Crear un Projecto

Instalar Angular CLI global:

```
npm install -g @angular/cli
```

Crear espacio de trabajo:

```
ng new [PROJECT NAME]
```

Iniciar la aplicación:

```
cd [PROJECT NAME]
ng serve
```
